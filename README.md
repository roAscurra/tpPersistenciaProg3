# Requisitos para la ejecución del proyecto:
1. JVM (version 17)
2. Gradle
3. H2 database
# Características de ejecución
1. Una vez descargado el proyecto, ejecutarlo y luego abrir http://localhost:8080/h2-console/ en el navegador. 
2. Checkear la conexión y si es exitosa, conectarse.
3. Luego podrá ver las tablas con sus relaciones y demás.
